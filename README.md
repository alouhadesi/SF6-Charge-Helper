\# 街霸6 蓄力助手 (SF6 Charge Helper)



一个基于 Python/Pygame 开发的《街霸6》蓄力练习辅助工具。

提供可视化的蓄力时间轴、残留显示、以及精确的帧数判定。



\## ✨ 主要功能

\* \*\*可视化时间轴\*\*：直观显示蓄力状态、按键输入和残留时间。

\* \*\*多模式支持\*\*：

&nbsp; \* 练习模式：针对 \[4]6 (横向) 和 \[2]8 (纵向) 的专项练习。

&nbsp; \* 自由模式：自动检测左右方向蓄力，支持 1P/2P 位置无缝切换。

\* \*\*精准判定\*\*：

&nbsp; \* 包含 6帧 输入宽容缓冲 (Input Buffer)。

&nbsp; \* 智能 SOCD 处理与报警。

&nbsp; \* 针对不同角色的帧数要求（如春丽 50F/30F）。

\* \*\*视觉与听觉反馈\*\*：

&nbsp; \* 蓄力完成变绿，松手残留显示青色拖尾。

&nbsp; \* 柔和的正弦波提示音效。

\* \*\*外设支持\*\*：

&nbsp; \* 支持全键盘自定义按键。

&nbsp; \* 支持 Xbox / PS 手柄输入。



\## 🛠️ 如何运行源码

1\. 安装 Python 3.x

2\. 安装依赖库:

&nbsp;  `pip install -r requirements.txt`

3\. 运行程序:

&nbsp;  `python main.py`



\## 🎮 操作说明

\* \*\*键盘\*\*：默认 WASD 移动，UIO 拳，JKL 脚。可在设置页修改。

\* \*\*手柄\*\*：即插即用，支持摇杆与十字键。

\* \*\*1P/2P 切换\*\*：在练习模式右上角点击按钮切换；自由模式下自动识别。
